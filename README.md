# 🏋️ AI Gym Coach

An AI-powered fitness assistant that helps users perform exercises correctly by analyzing body posture in real-time.

---

##  Features

*  Real-time pose detection using computer vision
*  AI-based form correction and feedback
*  Tracks exercise performance
*  (Optional) Voice guidance support
*  User-friendly interface with Streamlit

---

## 🛠️ Tech Stack

* **Python**
* **OpenCV**
* **MediaPipe**
* **NumPy**
* **Streamlit**

## ⚙️ Installation

1. Clone the repository:

git clone https://github.com/your-username/Ai-gym-coach.git
cd Ai-gym-coach

2. Create virtual environment:


python -m venv venv
venv\Scripts\activate


3. Install dependencies:


pip install -r requirements.txt




## ▶️ Run the Project


streamlit run app.py
